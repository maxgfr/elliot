De base, on a le fichier "headerTest.html" créé par le fabuleux Vavl'.

De "headerTest.html", j'ai créé une copie php nommée "new_headerTest.php" afin de créer les masques(à reporter sur toutes les pages):
	- Le header nommé "header.php"
	- La sidebar étendue nommée "sidebar.php"
	- La sidebar contractée nommée "iconbar.php"
// Le nom des masques commencent par une minuscule. //

Ensuite, lorsqu'on clique sur les icônes, on doit arriver sur une nouvelle page. On définit :
	- Les notifications nommés "Notification.php"
	- Le profil nommé "Profile.php"
	- La déconnexion nommée "Deconnection.php"
	- Le plan de la maison nommé "Homeplan.php"
	- Le dashboard nommé "Dashboard.php"
	- La gestion des capteurs/actionneurs nommée "Sensor.php"
	- Le support nommé "Support.php"
// Le nom des pages commencent par une majuscule. //




********************************TL;DR*******************************************
On a donc comme architecture, en partant de Homeplan.php comme page d'accueil :

Homeplan.php (à partir du header)
	--  Notification.php
	-- 	Profile.php
	--	Deconnection.php


Homeplan.php (à partir de la sidebar)
	--	Homeplan.php (via le logo Elliot)
	--	Dashboard.php
	--	Sensor.php
	--	Support.php
A noter que cette architecture devra également être accessible sur toutes les autres pages grâce aux masques contenant les liens dans leur code.
